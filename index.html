London Travel


In this project I took an older challenge from my bootcamp where I worked in a group to transform data into a database. I then took that data and set up a SQLAlchemy connection to PSQL, this deviated from how the group originally decided to use CSV files to transfer the data to PSQL. After setting up the connection and creating the tables from the data in the pandas dataframes, I then created four new tables -- using group bys -- for my analysis to be done on. Sending the new tables into my jupyter notebook, I created four new plots, visualizing the data in a much nicer format.
